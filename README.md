# NameStruct Extension

NameStruct is an extension that allows you to load custom structured JSON templates for enhanced naming and structuring capabilities.

## Loading First Template and Setup

1. **Get the JSON File URL**
   - Navigate to GitHub and open the raw file of either `example1.json` or `example2.json`
   - Copy the URL of the raw json file (`Raw` button is in the top right next to download)
      - URLs for raw file will start with `raw.githubusercontent.com`

2. **Configure NameStruct**
   - Open the NameStruct extension in your browser
   - Click the gear icon (⚙️) to access settings
   - Paste the copied URL into the text field
   - Click the "Load JSON" button

3. **Verify Successful Load**
   - If the JSON is valid, you'll see a "loaded" confirmation with no error messages
   - Invalid JSON files will display appropriate error messages

4. **Use Your New Templates**
   - Press the back button to return to the main interface
   - Review, search, and use your newly loaded templates

## Troubleshooting

- Ensure the JSON file is properly formatted as a JSON Array
- Verify the raw GitHub URL is accessible
- Check that the JSON structure matches the expected format
